# Fiction Library

## Цель проекта
Веб-приложение электронной библиотеки с автоматической синхронизацией книг из Telegram, каталогом, поиском и админ-панелью для управления контентом.

## Границы MVP
- Входит: каталог книг, поиск/фильтрация, аутентификация, Telegram-синхронизация, админ-панель, FB2 читалка, система серий
- НЕ входит: мобильные приложения, интеграция с внешними сервисами (кроме Telegram), многоязычность, монетизация
